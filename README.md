# 🎣 Карта рыболовных акваторий России

Интерактивная карта с отмеченными пинами трёх ключевых рыболовных регионов России:  
**Рузское водохранилище (МО)**, **озёра Карелии** (Ладога, Онега, Выгозеро, Сегозеро, Топозеро) и **Астраханская зона** (дельта Волги).

Выберите акваторию на карте или из выпадающего списка — и узнайте, какая рыба активно клюёт в выбранный месяц.

## 📌 Функционал

- **Интерактивная карта** на базе Яндекс.Карт (JavaScript API).
- **Цветные маркеры** с эмодзи 🎣 — каждая акватория выделена.
- **Фильтр по месяцам** — показывает активность клёва для каждой рыбы.
- **Детальная карточка** рыбы: рекорд, лучшая наживка, время клёва, советы.
- **Адаптивный дизайн** — работает на телефонах, планшетах и десктопах.

## 🗺️ Акватории

| Регион | Отмеченные точки |
|--------|------------------|
| **Московская обл.** | Рузское водохранилище |
| **Карелия** | Ладожское озеро, Онежское озеро, Выгозеро, Сегозеро, Топозеро |
| **Астраханская обл.** | Дельта Волги |

Для каждой акватории представлен список рыб с помесячной активностью (0–3).

## 🛠️ Технологии

- **HTML5 / CSS3** – вёрстка и анимации.
- **JavaScript** – логика, работа с картой, фильтрация.
- **Яндекс.Карты (JavaScript API 2.1)** – отображение карты и маркеров.
- **Chart.js** – графики на странице «Климат Арктики» (опционально).
- **XLSX.js** – экспорт данных в Excel.

## 🔑 Настройка API-ключа

Для работы карты необходимо **получить собственный API-ключ Яндекс.Карт**:

1. Перейдите в [кабинет разработчика Яндекс](https://developer.tech.yandex.ru/services/).
2. Выберите **«JavaScript API»** в разделе «Карты».
3. Создайте новое подключение (бесплатный тариф).
4. Скопируйте сгенерированный ключ.
5. В файле `index.html` найдите строку:
   ```html
   <script src="https://api-maps.yandex.ru/2.1/?apikey=ваш_ключ_яндекс_api&lang=ru_RU" type="text/javascript"></script>
